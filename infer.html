Infer

The infer module contains the main logic for inferring ML-ready data types.

infer.calculate_sample_size(population_size, margin_error=0.01, confidence_level=0.995, sigma=0.5)[source]

Calculate the minimal sample size to use to achieve a certain margin of error and confidence level for a sample estimate of the population mean. Inputs ——- population_size: integer

Total size of the population that the sample is to be drawn from.

margin_error: number

Maximum expected difference between the true population parameter, such as the mean, and the sample estimate.

confidence_level: number in the interval (0, 1)

If we were to draw a large number of equal-size samples from the population, the true population parameter should lie within this percentage of the intervals (sample_parameter - e, sample_parameter + e) where e is the margin_error.

sigma: number

The standard deviation of the population. For the case of estimating a parameter in the interval [0, 1], sigma=1/2 should be sufficient.

Provided the column data, define its data type and data subtype.

Parameters:

  • data (Union[ndarray, list]) – an iterable containing a sample of the data frame

  • full_data (DataFrame) – an iterable containing the whole column of a data frame

Returns:

type and type distribution, we can later use type_distribution to determine data quality

NOTE: type distribution is the count that this column has for belonging cells to each DATA_TYPE

infer.infer_types(data, pct_invalid, seed_nr=420, mp_cutoff=10000.0)[source]

Infers the data types of each column of the dataset by analyzing a small sample of each column’s items.

Return type:

TypeInformation

Inputs

datapd.DataFrame

The input dataset for which we want to infer data type information.

pct_invalidfloat

The percentage, i.e. a float between 0.0 and 100.0, of invalid values that are accepted before failing the type inference for a column.

seed_nrint, optional

Seed for the random number generator, by default 420

mp_cutoffint, optional

How many elements in the dataframe before switching to parallel processing, by default 1e4.
